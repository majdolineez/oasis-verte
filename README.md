# Oasis Verte 🌿

Site vitrine élégant et responsive dédié au secteur paysagisme. 

## 🎯 Objectif
Concevoir une interface web fluide et immersive valorisant l'activité de l'entreprise tout en garantissant une expérience utilisateur optimale.

## 🛠️ Stack Technique
* **Front-end :** HTML5, CSS3, JavaScript.
* **Design :** Intégration responsive (Mobile-first).
* **Outils :** Git, GitHub Pages.

## ✨ Points forts
* **Design UI :** Interface moderne, épurée et cohérente.
* **Responsive :** Navigation fluide sur tous les supports (Desktop, tablette, mobile).
* **Performance :** Chargement rapide et code sémantique.
